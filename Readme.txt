Welcome to Remote Plugin
Follow the steps to install.

1. Copy "Silkroad Remote.py" to phBot Testing\Plugins directory
2. In order to run the plugin, you need to install some libraries.
	All libraries that needed are in "python38" file.
	
	*Option1*
		Copy it to Testing\Plugins directory.
		Replace the files in the destinatoin.


	*Option2*
		Go to Testing\Plugins directory.	
		Open "python38\Lib\site-packages"
		Also open files that you downloaded. go to same directory: "pyhon38\Lib\site-packages"
		Copy files that you downloaded to "python38\Lib\site-packages"
